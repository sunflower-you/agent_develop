
# CC Switch
功能：管理各种 模型 API 的配置
链接：https://github.com/farion1231/cc-switch?tab=readme-ov-file

## 安装
### macos
```
brew tap farion1231/ccswitch
brew install --cask cc-switch
# update
brew upgrade --cask cc-switch
```

### linux
下载链接：https://github.com/farion1231/cc-switch/releases
安装命令：
```
sudo dpkg -i '/home/yyds/下载/CC-Switch-v3.11.1-Linux-x86_64 (1).deb'
./*APP
```
