

## 多agent
1. 添加新的隔离智能体
```
openclaw agents add main # 
openclaw agents add product_manager   # 收集需求
openclaw agents add designer  # 前段设计
openclaw agents add architect  # 架构师
openclaw agents add coder  # 开发人员
openclaw agents add great presenter

openclaw agents add tester   # 测试人员
openclaw agents add publisher  # 部署发布人员

openclaw agents add video_production  # 影视制作
openclaw agents add writer_producer # 编剧制片人

openclaw agents add writer  # 作家

```


| agent 名字        | 职责           | 主要职责 |
|------------------|----------------|----------|
| main             | AI助手         | 协调统筹全局任务、理解用户需求、拆解任务并分配给各Agent、汇总最终结果 |
| product_manager  | 产品经理       | 梳理需求、定义功能范围、输出PRD文档、确定优先级、协调设计与技术方向 |
| designer         | 设计师         | 负责UI/UX设计、交互流程设计、输出设计稿（原型/视觉）、确保用户体验一致性 |
| architect        | 架构师         | 设计系统架构、技术选型、模块划分、接口规范定义、保障系统扩展性与稳定性 |
| coder            | 开发人员       | 根据设计和架构实现功能、编写代码、接口开发、修复bug、保证代码质量 |
| tester           | 测试人员       | 编写测试用例、功能测试/回归测试、发现并跟踪bug、验证系统稳定性 |
| publisher        | 部署人员       | 构建部署流程（CI/CD）、环境配置、上线发布、监控系统运行状态 |
| video_production | 影视制作       | 负责产品宣传视频制作、剪辑、动画、视觉包装，提升产品传播效果 |
| writer           | 编剧制片人     | 编写产品故事、宣传文案、脚本策划、品牌叙事设计，增强产品表达与传播力 |
